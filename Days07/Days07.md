함수(function)란 하나의 특별한 목적의 작업을 수행하기 위해 독립적으로 설계된 코드의 집합

# 1. 함수로 코드를 재사용하는 방법

함수로 재사용되는 코드를 작성한다면 필요할 때마다 함수만 호출하면 되므로 재사용성이 증가합니다.

함수는 `func` 키워드를 작성하고 뒤에 원하는 함수명을 선언한 후 `()` 를 붙인 후 `{}` 블록안에 동작할 코드를 작성합니다.

함수가 동작할 때 필요한 데이터를 전달할 때 매개변수를 통해서 전달할 수 있습니다.

매개변수는 항상 함수를 생성할 때 나열된 순서대로 전달해야합니다.

⚠️ 함수내에서 생성한 모든 데이터는 함수가 완료되면 모두 제거됩니다.

<aside> 💡 함수에 전달하는 값을 인자, 함수 내부에서 받는 값을 매개변수라고 합니다.

</aside>

# 2. 함수에서 값을 반환하는 방법

함수에서 데이터를 다시 반환하고 싶은 경우 `->` 를 작성하고 `{` 앞에 반환할 데이터자료형을 작성합니다. 그리고, `return` 키워드를 통해 원하는 데이터를 반환할 수 있습니다.

-   함수가 값을 반환하고 코드라인수가 한줄만 있는 경우 `return` 키워드를 생략할 수 있습니다.

# 3. 함수에서 여러값을 반환하는 방법

함수에서 반환할 데이터자료형을 컬렉션타입을 사용하여 반환하면 여러 값을 반환할 수 있습니다.

> 튜플 반환 예시

```swift
func getUser() -> (firstName: String, lastName: String) {
    (firstName: "Soo", lastName: "Kim")
}

let (firstName, lastName) = getUser()
print("Name: \\(firstName) \\(lastName)")

// Name: Soo Kim

```

# 4. 매개변수 활용하는 방법

함수에게 데이터를 전달할 때 매개변수명을 작성할 때 여러가지 옵션을 사용할 수 있습니다.

-   `외부 매개변수명` : 함수를 호출할 때 사용하는 매개변수 이름입니다.
-   `내부 매개변수명` : 함수가 동작하는 곳에서 사용하는 매개변수 이름입니다.

1.  함수를 호출할 때 외부매개변수명을 무시하고 싶은 경우 `_` 를 사용할 수 있습니다.
2.  외부매개변수명과 내부매개변수명을 다르게 하여 사용할 수 있습니다.
